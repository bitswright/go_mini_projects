# Project 0: Hello CLI

## Requirements
### Functional Requirements
- The program should print a greeting message.
- If a name is passed as a command-line argument, the program should greet that user.
- If no name is provided, it should print a default greeting.
- Example Usage
    ```bash
    $ go run main.go
    Hello, World!
    
    $ go run main.go Harish
    Hello, Harish!
    ```
